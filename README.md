# Train Ticket Booking System (C)

A modular Train Ticket Booking System developed in C using layered architecture and file-based persistence.

## Features

- Admin Login
- Passenger Registration
- Passenger Login
- Train Management (CRUD)
- Train Search
- Ticket Booking
- Ticket Cancellation
- Booking History
- File-based Data Persistence

## Tech Stack

- C11
- GCC / Clang
- Makefile
- Modular Programming
- File Handling
- Git & GitHub

## Project Structure

```
src/
include/
data/
tests/
docs/
assets/
build/
```

## Build

```bash
make
```

## Run

```bash
./build/bin/train_booking
```
